## Next JS Upload Image with UploadThing

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Set key in .env
```bash
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
